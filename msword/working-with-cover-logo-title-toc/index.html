Working with Covers, Titles, Logos, Copyright and Table of Contents

This section covers working with document covers, titles, logos, copyright and inserting and updating a table of contents.

Cover or Title Page

All documents have a cover or a title page. It contains the logo, the title of the document, and the release or publication month and year. This is shown in the following figure.

Note: The cover or title page does not have a header or footer.

Working with Logos

All documents must contain a logo wherever applicable. These logos are designed in different sizes and their usage depends on how and where they are used in the document. For example, the size of the logo is different on the title page than on the pages of the document.

When working with logos:

  • Insert the correct logo.
  • Use the appropriate logo dimensions on the cover and pages as defined in your style guide. For example, the size of the logo in the cover page should be 229X86 pixels and so on.

    Note: The Table of Contents and the Index page should not contain the logo.

Specifying the Document Title

The document title must be self-descriptive and must clearly convey the purpose of the document.

When specifying the document title:

  • Use the correct product name, the version number, and the document type in the title. For example, ABC Software 1.0 Installation Guide. Here, ABC Software is the product, 1.0 is the version and Installation guide is the type of the document.

    Note: The above specifications may vary as per the project requirements.

  • Set the style to Heading 1

Release Month and Year

The release or publication month (and year) is given on the title page of the document just below the title. This allows you to track the changes in the documents. Each time you release the document, make sure that you update the month of release (and year where appropriate.

While writing the release month, do the following:

  • Use the <Month> <Year> format to write the release month and year. For example, March 2019.
  • Use the appropriate font face and size as defined in your style guide.
  • Set the paragraph spacing for the release month and year in the title page appropriately.

Adding the Copyright Page

The document copyright is defined in the copyright page and is added after the cover page. Depending on the format used by your organization, the information is divided into different sections.

While formatting of the copyright notice, make sure that: * You specify the correct copyright year and date range. The date range depends on the year of creation of the document to the year of updations.

Consider the following examples;

  • Copyright © 2005 - 2019 ABC Enterprises, LP. All rights reserved.
  • Copyright © 2006 - 2019 ABC Enterprises, LP. Inc. All rights reserved.
  • © 2019 ABC Enterprises, LP. All rights reserved.

Note: The year or date range that occurs on the copyright page also appears in the footers of the book.

  • You set the paragraph spacings as defined in your style guide.
  • You set the paragraph spacing of the first paragraph of the copyright statements section.
  • You set the Font Style, Font Name, and Font Size as defined in your style guide.
  • There MUST NOT be any header or footer on the copyrights page.

Working with a Document’s Table of Contents

A table of contents is a list of the topic titles in a document. You can use a table of contents to get an overview of the topics discussed in a document. This helps the reader to locate a topic or section within your document. In Microsoft Word, a table of contents is generated based on the heading styles that you apply to topics and sections of your document.

Note: It is important that you apply the correct heading styles to topics and sections that you want to appear in the table of contents. Always insert and update the table of contents right before you send documentation to peers and SMEs for reviews, or when releasing the guide to customers.

Inserting a Table of Contents

To insert a table of contents, go to References → Table of Contents → Custom Table of Contents.

The Table of Contents dialog box is displayed.

Do the following.

  • Select the Show page numbers and Right align page numbers check boxes.
  • From the Tab leader list, select the ….. tab leader.
  • Select Classic or any format of your choice from the Formats list.
  • Select a nesting level number from the Show levels list depending on your preferences.
  • Click the OK button. The table of contents is inserted.

Updating a Table of Contents

As you add content to your document, the page numbers of Headings, such as chapters, sections, and topics will constantly change. Therefore, update the table of contents every time you modify your document so that the correct headings and page numbers are reflected.

Note: Do not modify the entries in the table of contents manually. If you do so, changes are lost when you update the table of contents.

To update the table of contents:

  1. Place the cursor on the table of contents and press F9. The Update Table of Contents dialog box is displayed.

  2. Click Update entire table, and click the OK button.
