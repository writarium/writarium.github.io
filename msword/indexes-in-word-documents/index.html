Indexes in Microsoft Word

This section covers creating and inserting index markers. Building, updating, and troubleshooting indexes.

Creating an Index

Professional Indexers recommend using a variety of keywords and synonyms to create indexes. However, index entries are based on topic titles only. This is due to limited project times. Moreover, every released book has several ways in which readers can look up information of interest, including the tables of contents, keyword searches, and indexes. Few points to keep in mind while creating indexes are:

  • Index entries should be in lower case, with the following exceptions:
    • Proper nouns
    • Acronyms and abbreviations
  • Quotation marks appear before and after the words that you want to list in the index.
  • There are no spaces between the words and the colons. You must avoid spaces to prevent the index entries from becoming misaligned in the index.
  • Index markers are added directly after the title in each section. This makes it easier to find them, and prevents them from being accidentally deleted.
  • Keep the index complete and up-to-date.
  • While creating an index, think of the questions that various levels of users might ask.
  • Always check the spellings of the index entries by running a spell checker with the Show All feature turned on.
  • You can indent the index entries by using a colon (:). However, the level of indent must be to a maximum of 3 indents. For example, enter { XE “How to:insert:index markers” }.
  • Divide a primary entry in to sub entries.
  • Group related sub entries under appropriate primary entries

Inserting Index Markers

The index markers are defined based on the headings. As a convention, there are at least two index makers per heading. However, the formats in which these index entries are given differ for each heading type.

  • H2 to H4 headings: There are generally two markers added per title. For one index entry, the verb is followed by a noun while for another its vice versa. This enables the reader to look up topics in a couple of ways. For example, enter { XE “deleting:customer records” }{ XE”customer records:deleting” }. A reader can look for both deleting and customer records to find topics of interest.

  • For H5 headings: Include two index markers. For procedures, use the How To: <verb> format, for example. However, for the second entry you can use the verb followed by a colon and How to or simply use the verb. The basic aim is to provide a quick reference to the user in an organized manner. For example, { XE “How to:insert index markers” }{ XE”insert index markers:how to” }

To insert index markers:

  1. Open the document is which you are working, and make sure you can view all hidden text.

    To view hidden text, press the combination of Alt + F9 keys on your keyboard.

  2. Place the cursor after the title.

  3. On the References tab, click Mark Entry.

  4. The Mark Index Entry dialog box appears.

  5. Complete the index entry. For example, enter Inserting index markers The index entry is marked next to the title and is labelled as {XE”inserting:index markers”} code.

    As mentioned above, to view the XE field codes, press the combination of Alt + F9 keys on your keyboard.

  6. Repeat steps 2 to 5 until you are finished with all the titles in the document.

Building an Index

After you have marked all the index entries, you can start building the index.

  1. To ensure that the document is paginated correctly, hide the field codes and hidden text. If the XE (Index Entry) fields are visible, click Show/Hide button ¶ on the standard tool bar.
  2. Click where you want to insert the finished index. This is probably at the last page of the guide.

  3. On the References tab, click Insert Index. The Index dialog box appears.

  4. Select Classic from the Formats list.

  5. In the Type area, click Indented if needed.

  6. Select 2 on the Columns list.

  7. Click the OK button. The index is inserted.

  8. Select all text and change to a font face, color, and size of your choice.

Updating Indexes

If you modify the document, you need to update the index. Do not modify the entries in the finished index manually. If you do so, changes are lost when you update the index. Therefore, it is recommended to edit the text in the index markers, and not in the index.

To update an index:

  1. Click to the left of the field.
  2. Press F9. The index is updated.
  3. Select all index text and change the font face, color, and size.

Troubleshooting Indexes

When you insert an XE field code, a space may be inserted in the field code between the words and the colon (:) as shown below.

The presence of this space causes indenting problems in the index. To resolve this, click the Show/Hide button ¶ on the standard tool bar and remove the space between the word and colon (:).
